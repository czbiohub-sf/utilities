Example commands

You can run the pipeline using nextflow run.

View help

You can use --help as a parameter to get an overview of the possible parameters.

nextflow run czbiohub/utilities \
  -r main_build -latest \
  -main-script target/nextflow/multiomics/cellranger_arc_count/main.nf \
  --help

Run command

Example of params.yaml 
nextflow run czbiohub/utilities \
  -r main_build -latest \
  -profile docker \
  -main-script target/nextflow/multiomics/cellranger_arc_count/main.nf \
  -params-file params.yaml
Note

Replace -profile docker with -profile podman or -profile singularity depending on the desired backend.

Argument group

Arguments

Name Description Attributes
--libraries_path Path to the libraries file, required, example: "/path/to/libraries"

--output Path to the output directory file, required, example: "/path/to/directory"
--run_id The run ID string, required
--peaks Path to the bed file to use for peak calling file, example: "/path/to/bed"
--neurips Unknown boolean, default: FALSE
